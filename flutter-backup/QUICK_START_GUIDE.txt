STEP 1: Flutter SDK (15 minutes)
---------------------------------
Download: https://docs.flutter.dev/get-started/install/windows
Extract to: D:\Projects\flutter
Add to PATH: D:\Projects\flutter\bin
Verify: flutter --version

STEP 2: Android Studio (60 minutes)
------------------------------------
Download: https://developer.android.com/studio
Install with Android SDK
Open SDK Manager:
  - Install Platform 36
  - Install Build-Tools 36.0.0
  - Install Emulator
Set environment: ANDROID_HOME = C:\Users\[user]\AppData\Local\Android\Sdk
Accept licenses: flutter doctor --android-licenses
Install plugins: Flutter, Dart

STEP 3: Visual Studio Build Tools (45 minutes)
-----------------------------------------------
Download: https://visualstudio.microsoft.com/downloads/
Select: Desktop development with C++
Include: Windows 10 SDK

STEP 4: Firebase CLI (10 minutes)
----------------------------------
Install: npm install -g firebase-tools
Login: firebase login
Install FlutterFire: dart pub global activate flutterfire_cli
Add to PATH: %USERPROFILE%\AppData\Local\Pub\Cache\bin

STEP 5: Clone Project (5 minutes)
----------------------------------
git clone [your-pharmapp-mobile-repo-url]
cd pharmapp-mobile

STEP 6: VS Code Extensions (10 minutes)
----------------------------------------
Essential extensions to install:
  - Dart (dart-code.dart-code)
  - Flutter (dart-code.flutter)
  - Claude Code (anthropic.claude-code)
  - GitLens (eamodio.gitlens)
  - Error Lens (usernamehw.errorlens)

STEP 7: Configure Firebase (20 minutes)
----------------------------------------
cd pharmacy_app
flutterfire configure --project=mediexchange

cd ../courier_app
flutterfire configure --project=mediexchange

cd ../admin_panel
flutterfire configure --project=mediexchange

cd ../pharmapp_unified
flutterfire configure --project=mediexchange

STEP 8: Install Dependencies (15 minutes)
------------------------------------------
cd shared && flutter pub get
cd ../pharmacy_app && flutter pub get
cd ../courier_app && flutter pub get
cd ../admin_panel && flutter pub get
cd ../pharmapp_unified && flutter pub get

STEP 9: Verify Installation (15 minutes)
-----------------------------------------
flutter doctor -v          # All should be green
cd pharmacy_app
flutter build apk          # Test Android build
flutter run -d chrome --web-port=8084  # Test web run
